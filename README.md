# Battlestar Galactica

[TTS Workshop Link](https://steamcommunity.com/sharedfiles/filedetails/?id=2064475872)

This mod is a scripted version of the wonderful Battlestar Galactica board game by Fantasy Flight Games. The entire setup process is scripted. The script also adds some additional functionality listed below. The Pegasus, Exodus and Daybreak expansions are all supported, as are all official variants and a number of unofficial variants.
